# Typing-speed-analyser
A dynamic typing system that continuously generates diverse, real-world content from Wikipedia while maintaining a fluid and responsive user experience.

- • Dynamic Content Generation — Fetches and filters Wikipedia data to provide varied, non-repetitive passages
- • Real-Time Performance Tracking — Calculates WPM and accuracy instantly as the user types
- • Seamless Sentence Streaming — Appends new sentences at the right moment to eliminate pauses
- • Adaptive Flow — Preloads upcoming content to ensure zero-latency transitions
- • User-Focused Design — Clean UI with multiple modes, difficulty levels, and gamified interactions
